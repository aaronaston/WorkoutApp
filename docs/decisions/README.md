# Decisions

This folder tracks architecture and design decisions.
Decision status (`Accepted`, etc.) reflects design intent, not guaranteed feature completion.
Use `docs/current-state.md` and `docs/gap-analysis-current-vs-target.md` for implementation status.

## Subfolders
- adr/ : Architecture Decision Records
- ddr/ : Design Decision Records
- registries/ : ADR/DDR indexes (one per type)

# Hinge/Push A

## Prep
- Deep Lunge with Sky (Thoracic) Reaches — 1 x 10, per side
- A-Frame Shoulder Opener Pulses + Ankle Touches — 1 x 10, alternating
- Adductor Rocks — 1 x 10, per side
- Deadbug Pallof Press (or Regular Deadbug) — 2 x 30s
- Banded Scap Push Ups — 2 x 30s
- Superman Hold - Lower Only — 2 x 30s; brace core and press hips to floor

## Push
- Push Up Negative with Deficit — 3 x 6
- Single Leg Posted RDL — 3 x 10, per side
- Weighted Side Plank — 3 x 30s, per side

## Push: Intervals 30s / 15s (option to add rest station)
- Chin Up Prep — 3-4 x 30s
- Sumo Pulses — 3-4 x 30s (hold weight as chest)
- Dumbbell Strict Press — 3-4 x 30s
- Banded March - Hip Flexor — 3-4 x 30s

## Ponder / Cool-Down
- Seated Butterfly Stretch - Weighted — 1 x 90s
- Childs Pose with Reaches — 1 x 60s

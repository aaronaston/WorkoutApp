# Squat/Pull B

## Prep
- Side Lying Thoracic Reaches — 1 x 8-10, per side
- Laying Windshield Wiper - Stretch — 1 x 10, alternating slowly
- Laying Pelvic Tilt — 1 x 15
- Single Arm Scap Pull - Feet Assisted — 2 x 8, per side
- Monkey - Foundational — 2 x 5, per direction
- Superman Lifts - Lower Only, Single Leg — 2 x 10, per side; core braced to isolate glutes

## Push 1
- Pause Banded Goblet or Barbell Back Squat — 3 x 6-8; 1s count down, 3s pause, quick up, 2s pause at top; noted set: 20# DB x8

## Push 2: Intervals 30s on 15s off
- Top of Pull Up Hold — 5 x 30s; assistance as needed (band or feet); note: 20-25 sec unassisted on first set, then 15-20, then assisted
- Rear Elevated Split Squat - Pulses — 5 x 30s, per side; hold weight if moving well
- Side Plank with Leg Raise — 5 x 30s, per side; or static single arm farmers hold if less painful as long as heavy
- Rest — 5 x 30s

## Ponder / Cool-Down
- Elevated Pigeon Stretch — 1 x 90s, per side
- Banded Lat Opener — 1 x 60-90s, per side

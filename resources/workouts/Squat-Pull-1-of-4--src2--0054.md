# Squat/Pull (1/4)

## Prep
- Chin Tucks — 1 x 12-15
- 90/90 Shinbox (Segmented) — 1 x 10; option to elevate hips sitting on yoga block
- Thread-the-Needle with Sky Reach — 1 x 10, per side
- Cobra Yoga Pose — 1 x 10; gentle lift and lowers
- Single Leg Glute Bridge — 2 x 8, per side
- Deadbug - Contralateral — 2 x 10; alternating with breath slowly
- Banded Glute Side Kicks — 2 x 12, per side

## Practice
- Hip ER PAILs/RAILs — 1 x 90s+, per side

## Push 1
- Shrimp Squat Variation: Negative — 3 x 5, per side
- Feet-Assisted Pull Ups (OR 3x Lat Pull Downs) — 3 x 5

## Push 2
- Chest Supported Dumbbell Rows — 3 x 12
- Banded Goblet Squat — 3 x 12; noted weight: 20#

## Push 3: EMOM 6 “Finisher”
- Wall Squat — 2 x 40s; hold weight for added challenge
- Monster Side Steps — 2 x 40s
- Banded Pull Aparts (Palms Up) — 2 x 40s

## Ponder / Stretch
- Foam Rolling - Quads/IT — 1 x 45s+, per side
- Lying Glute Stretch — 1 x 45s+, per side

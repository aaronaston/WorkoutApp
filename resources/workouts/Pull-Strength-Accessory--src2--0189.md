# Pull Strength (Accessory with Bike, Walk, Etc)

## Prep
- Kneeling Thoracic Rotations with Wall — 1 x ~60s per side
- Froggy Hip Opener Rocks — 1 x ~60s
- A-Frame Shoulder Opener Pulses + Ankle Touches — 1 x ~60s
- Hollow Body Flies — 2 x 15 (bend knees if needed)
- Single Arm Scap Pull – Feet Assisted — 2 x 10 per side
- Prone Shoulder Swimmers — 2 x 5

## Practice: Pull Up Negatives
- Pull Up Negative with Jump from Box — 5 x 5; rest 60–90s between

## Extra Stretching
- Thoracic Foam Rolling — 1 x 90s
- Lacrosse Ball – Trap Smashing — 1 x 90s
- Passive Squat Hold — 1 x 90s; play with position to see where hip pain is alleviated

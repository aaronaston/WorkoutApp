# Hinge/Push B

## Prep
- Deep Lunge with Sky (Thoracic) Reaches — 1 x 10, per side
- A-Frame Shoulder Opener Pulses + Ankle Touches — 1 x 10, alternating
- Adductor Rocks — 1 x 10, per side
- Laying Pelvic Tilt — 2 x 30s
- Bird Dog - Beginner — 2 x 30s, per side
- Banded Glute Bridge — 2 x 30s

## Push
- Single Leg Deadlift - Back Leg Wall Supported — 3 x 10-12, per side
- Hex Press — 3 x 10-12; build in weight if moving well
- Kneeling Banded Woodchoppers — 3 x 10-12, per side (make sure final reps are hard)

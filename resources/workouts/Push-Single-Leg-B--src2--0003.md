# Push/Single Leg B

## Prep
- Banded Dislocates — 1 x 15-20
- Thread-the-Needle with Sky Reach — 1 x 8-10, per side
- 90:90s with Raise — 1 x 10, alternating
- Laying Pelvic Tilt — 1 x 10-15, slow with breath
- Scap CAR w/ Yoga Block — 2 x 6-8, per direction
- Side Plank Clamshells with Band — 2 x 15, per side
- Glute Bridge with Adductor Squeeze — 2 x 15

## Practice: Balance/Stability (3 Minutes As Desired)
- Single Leg Long Jumps — explore as desired (accumulate ~10-15 per side)
- Broad Jump

## Push 1
- Tempo Dumbbell Bench Press - Neutral — 3 x 8-10; 3-5s count down per rep
- Power Step Up — 3 x 8-10, per side
- Bird Dog DB Drag — 3 x 8-10, alternating; hover knees for added challenge

## Push 2
- B-Stance Hip Thrust — 3 x 30s, per side; or single leg glute bridge per side
- Active Bottom of Push Up Hold — 3 x 30s
- Seated Hip Flexor Lifts - Single Leg — 3 x 30s

## Ponder / Stretch
- Forward Splits — 1 x 90s, per side
- Childs Pose with Reaches — 1 x 90s

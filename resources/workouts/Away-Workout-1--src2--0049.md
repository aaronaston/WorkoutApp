# Away Workout #1

## Prep
- Wag-the-Tails — 1 x 6-8, per direction
- Kneeling Hip Opener with Side Bend/Reach — 1 x 6-8, deep breaths per side
- Adductor Rocks — 1 x ~15, per side
- Side Lying Thoracic Reaches — 1 x 10, per side
- Laying Pelvic Tilt with Breath — 1 x 10-15
- Bodyweight Glute Bridge — 2 x 15-20
- Prone W with Reach — 2 x ~6-8

## Push 1
- Reverse Lunge to Knee Drive — 3 x 10-12, per side; complete all before switching
- Elevated Push Up — 3 x 12-15
- Side Plank — 3 x 40s, per side; drop bottom knee if needed
- B-Stance Hip Thrust — 3 x 12-15, per side; if easy add pause at top; can use chair, couch, etc
- Wall Slides — 3 x 8-10
- Plank Step Outs — 3 x 20

## Ponder / Stretching
- Childs Pose with Reaches — 1 x 60-90s
- Laying Butterfly OR Feet Elevated Diaphragm Breathing — 1 x 60-90s; option to add pillows under knees

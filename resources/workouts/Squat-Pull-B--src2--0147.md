# Squat/Pull B

## Prep
- Side Lying Thoracic Reaches — 1 x 8-10, per side
- Laying Windshield Wiper - Stretch — 1 x 10, alternating slowly
- Laying Pelvic Tilt — 1 x 15
- Hip Flexion- PRH — 2 x 30s, per side
- Wall Slides — 2 x 8
- Monkey - Foundational — 2 x 5, per direction
- Side Plank Clamshells with Band — 2 x 12-15, per side

## Push 1
- Pull Up Prep — 3 x 6-8; more volume, use small assist to maintain form (6 reps assisted noted)
- Banded Goblet or Back Squat — 3 x 8-10 (10 reps, 20# DB noted)

## Push 2: Intervals 30s on / 15s off
- Rear Elevated Split Squat - Pulses — 4 x 30s, per side; hold weight if moving well
- Prone Dumbbell Rows or TRX Rows — 4 x 30s
- Weighted Side Plank — 4 x 30s, per side
- Rest — 4 x 30s

## Ponder / Cool-Down
- Elevated Pigeon Stretch — 1 x 90s, per side
- Banded Lat Opener — 1 x 60-90s, per side

# Gym Workout - Squat/Pull

## Prep
- Side Lying Thoracic Reaches — 1 x 60–90s per side
- 90/90s with Raise — 1 x 60–90s alternating
- Low Banded Rows — 3 x 15
- Modified Bird-Dog — 3 x 5 per position, per side
- Side Plank Clamshell (With or Without Band) — 3 x 10–15 per side

## Practice: Chin Up Negatives
- Chin Up Negatives — 4 x 3; rest 60–90s between sets

## Push
- Split Squats — 4 x 12 per side
- Single Arm Dumbbell Rows — 4 x 12 per side
- Back Plank — 3–4 x 30s
- Single Arm Farmer's Carry Hold — 3–4 x 30s per side

## Ponder / Cool-Down Stretch
- Thoracic Foam Rolling — 1 x 90s
- Froggy Hip Opener Rocks — 1 x 90s
- Banded Lat Opener — 1 x 90s per side

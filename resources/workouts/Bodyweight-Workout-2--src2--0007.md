# Bodyweight Workout #2

## Prep
- Over-Unders — 2 x 4, per direction
- Bird Dog — 2 x 8, per side; focus on core + glutes
- Crab Table Top Raises — 2 x 8

## Push
- Reverse Lunge to Knee Drive — 3-4 x 12-15, per side; complete all before switching
- Forearm Plank Reaches — 3-4 x 12-15, per side alternating with control
- Feet or Knees Elevated Handstand — 3-4 x 40s

## Stretch
- Childs Pose with Reaches — 1 x 90s, alternating as desired
- Couch Stretch (Mobility WOD) — 1 x 60s+, per side

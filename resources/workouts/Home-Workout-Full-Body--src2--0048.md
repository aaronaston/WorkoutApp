# Home Workout (Full Body)

## Prep
- Cat-Cow — 1 x 8
- Thread-the-Needle with Sky Reach — 1 x 8, per side
- Laying Windshield Wiper - Stretch — 1 x 8; pausing 3-5s per rep
- Laying Hamstring Floss — 1 x 8, per side with 2-3s pause per
- Assisted Hip Airplanes — 1 x 10-15, per side
- Banded Pull Aparts (Palms Up) — 1 x 20; break up as needed

## Push 1

- Bench Supported Single Leg Deadlift — 3 x 10

## Push 2
- Push Up Negatives — 3 x 5
- Banded Goblet Squat — 3 x 10
- Dumbbell Cuban Rotations — 3 x 10

## Ponder / stretch
- Puppy + Lat Openers — 1 x 60-90s
- Elevated Pigeon Stretch — 1 x 60-90s, per side

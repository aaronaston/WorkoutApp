# Squat/Pull B

## Prep
- Side Lying Thoracic Reaches — 1 x 8-10, per side
- Laying Windshield Wiper - Stretch — 1 x 10
- Laying Pelvic Tilt — 1 x 15
- Hip Flexion- PRH — 2 x 30s, per side
- Single Arm Scap Pull - Feet Assisted — 2 x 8, per side
- Monkey - Foundational — 2 x 5, per direction
- Superman Lifts - Lower Only, Single Leg — 2 x 10, per side; core braced to isolate glutes

## Push 1
- Pull Up Prep — 3 x 6-8; more volume, use small assist to maintain form
- Banded Goblet or Back Squat — 3 x 8-10

## Push 2: Intervals 30s on / 15s off
- Rear Elevated Split Squat - Pulses — 4-5 x 30s, per side; hold weight if moving well
- Prone Dumbbell Rows or TRX Rows — 4-5 x 30s
- Weighted Side Plank — 4-5 x 30s, per side
- Rest — 4-5 x 30s

## Ponder / Cool-Down
- Elevated Pigeon Stretch — 1 x 90s, per side
- Banded Lat Opener — 1 x 60-90s, per side

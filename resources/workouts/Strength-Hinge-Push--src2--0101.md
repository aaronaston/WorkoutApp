# Strength - Hinge/Push

- Deep Lunge with Sky (Thoracic) Reaches — 1 x ~60s, per side
- Tail Wags — 1 x ~60s
- Modified Bird-Dog — 2 x 5, per limb, per side
- Single Leg Glute Bridge with Wall — 2 x 15, per side
- Standing Pelvic Drill — 2 x 40s, per side; resting as needed between

## Practice: Cossack Squat Flow (~3 Minutes)
- Squat Mobility Combo: Sumo + Cossack Squats — ~3 minutes

## Push

- Kneeling Banded Woodchoppers — 3 x 15, per side
- Single Leg Posted Romanian Deadlift — 3 x 10, per side

## Push: EMOM 40/20 focused on quality

- Kettlebell Sumo Deadlift — 3 x 40s; option to elevate KB/DB; move with control
- Plank Walk Ups — 3 x 40s; drop knees as needed

- Froggy Hip Opener Rocks — 1 x 90s
